# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-03

### Added
- Initial release of Hospital Management System
- Secure GUI login system with authentication
- Staff management module (CRUD operations)
- Patient management with medical history
- Ambulance fleet management
- Multi-role access control (Admin, Doctor, Nurse, Receptionist)
- Advanced search and filter functionality
- Data validation for all inputs
- File-based persistent storage
- Color-enhanced console interface
- Role-based access control

### Technical Details
- Built with C++ and SFML (for GUI)
- Object-Oriented Design with inheritance
- File-based database (.txt files)
- Cross-platform compatibility
- Multi-module architecture

### Known Limitations
- Console-based backend interface
- File-based storage (limited scalability)
- Single-user access
- Color support varies by platform
